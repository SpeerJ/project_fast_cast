# Actor Semantic Search Engine
A powerful semantic search engine leveraging GPT and Anthropic for actor data. It summarizes photos, compares embeddings, and provides accurate actor search capabilities.
## Features
- **Photo summarization:** Uses GPT and Anthropic to generate descriptions of actor photos.
- **Semantic search:** Finds actors based on natural language queries, improving search relevance.

## Future Features
- **Calendar Scheduling** Schedule remote or in person interviews
- **Video Store** Store and share actor videos
- **Actor Profiles** Store information about actors, tardiness to interviews, etc

## Technologies
- Ruby on Rails 8
- GPT and Anthropic APIs
- Vector embeddings for similarity comparison
- TailwindCSS for UI
- PostgreSQL / SQLite as database

## Usage
- Upload actor photos for automatic summarization.
- Perform semantic searches using natural language queries.
- Compare actor summaries based on embeddings for advanced filtering.
